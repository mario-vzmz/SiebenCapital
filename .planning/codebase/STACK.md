# Stack & Technologies

## Core Application
- **Frontend Framework**: React 18
- **Build Tool**: Vite v6
- **Language**: TypeScript / TSX
- **Styling**: Tailwind CSS (PostCSS)
- **Component Libraries**: Lucide React (Icons)
- **Markdown Render**: react-markdown (w/ remark-gfm)

## Backend / AI Integration
- **Relay Server**: Python (Flask) `relay.py`
- **AI Models**: Google Gemini (antigravity integration / native browser tools)
- **Networking**: ngrok (exposing local 5000 to internet webhook)

## Dependencies & Rationale
- Frontend focuses on high-speed modularity. Data is polled dynamically (no heavy state managers like Redux detected, reliance on React hooks).
